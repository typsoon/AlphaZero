Chess Puzzle Editor

Set up a position, record the expected move(s), export puzzle JSON for performance_evaluation/games/chess/<category>/

Position

Drag pieces from the trays above/below the board onto squares to edit the position. Drag a piece off the board to remove it.

Puzzle Metadata

en_passant is always exported as -1 and castling as [1,1,1,1,1,1] (nobody may castle) - this matches every hand-authored puzzle in the repo, since the engine's set_custom_state always resets move_count/en_passant_move to 0, so en-passant capture can never actually be legal for a puzzle loaded through /predict, and disabling castling keeps positions unambiguous.

Expected Move(s)

    Each move is encoded exactly like Chess::encode_action in engine/game/chess.cpp: action = (from_square*64 + to_square)*5 + promotion, from/to = row*8+col (row 0 = rank 8, col 0 = file a).

    Export

    After exporting, save the file into performance_evaluation/games/chess/<category>/<puzzle_name>.json, then run doit format:connect4_puzzles (it also formats chess puzzles) and doit validate_chess_puzzles to independently verify correctness with python-chess before trusting the puzzle.
